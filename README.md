## Keymap

| Key         | Action                 |
|-------------|------------------------|
| **w**       | Draw wire              |
| **i**       | Draw input node        |
| **o**       | Draw output node       |
| **t**       | Draw NPN transistor    |
| **p**       | Draw PNP transistor    |
| **Ctrl + L**| Open load dialogue     |
| **Ctrl + S**| Open save dialogue     |
